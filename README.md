# 🎓 Student Academic Performance Dashboard

An interactive Power BI dashboard designed to analyze student academic performance, payments, and course completion through dynamic and visually engaging insights.

---

## 📊 Overview

This project provides a comprehensive analysis of student data, helping to track performance, identify trends, and support better decision-making.

The dashboard combines KPIs, interactive filters, and multiple visualizations to deliver a clear and user-friendly analytical experience.

---

## 🚀 Features

### 📌 Key Performance Indicators (KPIs)
- Overall Average Payment  
- Completed Courses  
- Achievement of Target  
- Overall Average Grade  

---

### 🎯 Dynamic Filters (Slicers)
- Semester  
- Gender  

---

### 📈 Visualizations
- Completed Courses & GPA per student  
- Overall Average Grade by Class Name  
- Gauge chart for performance vs target  

---

### 🔍 Interactivity
- Fully dynamic visuals  
- Hover for detailed insights  
- Real-time updates based on filters  

---

## 🧠 Insights

Using this dashboard, you can:

- Identify top-performing students  
- Analyze course completion patterns  
- Compare subject performance  
- Track progress toward academic targets  
- Understand payment trends  

---

## 🛠️ Tools & Technologies

- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query (Data Cleaning & Transformation)  
- Data Modeling
